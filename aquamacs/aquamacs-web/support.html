Support

Users should contact Kevin Walzer on the Emacs-OS X mailing list hosted by Penn State University. The easiest way to subscribe is to send a message to: macosx-emacs-on@email.esm.psu.edu
No subject or body are needed. The list archives provide a useful resource for general issues of using Emacs on Mac OS X.

Bug reports should be sent to the Aquamacs bug reports list. The Aquamacs menu provides the means to file bug reports, which are automatically copied to this mailing list.

You can also use the Aquamacs wiki to add ideas, suggestions and tips for customizing and improving Aquamacs.
