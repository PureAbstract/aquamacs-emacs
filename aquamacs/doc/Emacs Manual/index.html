The Emacs Editor

Emacs is the extensible, customizable, self-documenting real-time display editor. This Info file describes how to edit with Emacs and some of how to customize it; it corresponds to GNU Emacs version 23.2.50.

For specific additions and deviations from Emacs as described in this manual, please refer to the Aquamacs manual.
For information on extending Emacs, see Emacs Lisp.
