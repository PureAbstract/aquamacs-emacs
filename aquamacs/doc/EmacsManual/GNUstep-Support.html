F.4 GNUstep Support

Emacs can be built and run under GNUstep however building is difficult and there are some limitations to functionality. In particular, it may be necessary to run `make bootstrap' with a plain X configuration, then `make clean' and `./configure --with-ns' followed by `make install'.

Currently CANNOT_DUMP is automatically enabled in GNUstep configurations, because the unex file(s) for GNUstep, mainly `unexelf.c', have not been updated yet with the “zone” code in and related to `unexmacosx.c'.
