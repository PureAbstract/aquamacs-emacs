39.1 Navigation

When in DocView mode, you can scroll the current page using the usual Emacs movement keys: C-p, C-n, C-b, C-f, and the arrow keys.

To display the next page, type n, <next> or C-x ] (doc-view-next-page). To display the previous page, type p, <prior> or C-x [ (doc-view-previous-page).

The <SPC> (doc-view-scroll-up-or-next-page) key is a convenient way to advance through the document. It scrolls within the current page or advances to the next. <DEL> moves backwards in a similar way (doc-view-scroll-down-or-previous-page).

To go to the first page, type M-< (doc-view-first-page); to go to the last one, type M-> (doc-view-last-page). To jump to a page by its number, type M-g M-g or M-g g (doc-view-goto-page).
