Enhanced Carbon Emacs Plugin Under Development

Enrico Franconi, developer of the old Enhanced Carbon Emacs distribution that had special support for LaTEX editing via its customizations for AucTEX mode, has revived ECE as an external plugin that can be installed on any Carbon Emacs,including Aquamacs. At this time, the ECE plugin is undergoing beta testing. The beta version can be downloaded from this location:

http://www.inf.unibz.it/~franconi/enhanced-carbon-emacs/ec-emacs-plugin-beta.dmg
