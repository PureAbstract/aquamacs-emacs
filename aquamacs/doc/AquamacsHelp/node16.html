The Right Look: Colors and Fonts for Frames and Modes

Aquamacs allows you to alter specific features of the current frame via functions in the Options menu. ``Font (this Frame)...'' lets you choose from the installed system fonts. ``Color Theme (this Frame)...'' will let you choose a pre-defined combination of colors.

These settings only apply to the current frame. To make them stick, enable ``Frame Styles / Auto Frame Styles'', then use the function ``Frame Styles / Use current style as default.'' Future frames will open by default with the new colors and fonts chosen. This applies to all visual frame-settings that a mode or you as a user have chosen using Emacs' configuration system.

Aquamacs Emacs also offers you to pick a style specific to the current mode. For example, you can use different settings when you are editing C or LaTeX, then when you are editing a text file. That is what the function ``Use current style for [current] mode'' is for. The setting will stick for newly opened frames or or whenever you newly use a given mode. ``Auto Frame Styles'' needs to be enabled.

Note that mode-specific styles will override any default style. If you'd like to use the same style for all frames, you should use ``Clear All Styles'' and then ``Use current style as default.''.

If you're more of an Emacs expert and you'd like to configure things manually by means of elisp or the customization buffer (``default-frame-alist'', etc.), just keep the ``Auto Frame Styles'' option turned off. You can then use the customization options ``default-frame-alist'' and ``initial-frame-alist'' as documented in the Emacs manual.

Finally, you may want to save your settings so they will stick even when you restart Aquamacs Emacs. To do so, use the function ``Save Options.''
