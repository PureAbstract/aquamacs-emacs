13.17 Changes--0.9.9c

  • Bugfix: Emacs-internal Input methods work again.

  • System-wide input methods (via the ``International'' preference panel) can now be used instead of the Emacs-specific ones. Use Options - Aquamacs Multilingual Environment - Use System Input Method to turn it on or off - it's switched on by default. Patch by Taiichi Hashimoto
