The Emacs Editor

Emacs is the extensible, customizable, self-documenting real-time display editor. This manual describes how to edit with Emacs and some of the ways to customize it; it corresponds to GNU Emacs version 24.3.50.

For specific additions and deviations from Emacs as described in this manual, please refer to the Aquamacs manual.
For information on extending Emacs, see Emacs Lisp.
