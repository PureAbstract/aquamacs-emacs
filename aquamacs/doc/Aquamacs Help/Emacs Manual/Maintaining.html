28 Maintaining Large Programs

This chapter describes Emacs features for maintaining large programs. If you are maintaining a large Lisp program, then in addition to the features described here, you may find the ERT (“Emacs Lisp Regression Testing”) library useful (see ERT).

  • Version Control: Using version control systems.
  • Change Log: Maintaining a change history for your program.
  • Tags: Go directly to any function in your program in one command. Tags remembers which file it is in.
  • EDE: An integrated development environment for Emacs.
  • Emerge: A convenient way of merging two versions of a program.
