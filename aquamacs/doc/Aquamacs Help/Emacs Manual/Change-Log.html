28.2 Change Logs

Many software projects keep a change log. This is a file, normally named ChangeLog, containing a chronological record of when and how the program was changed. Sometimes, there are several change log files, each recording the changes in one directory or directory tree.

  • Change Log Commands: Commands for editing change log files.
  • Format of ChangeLog: What the change log file looks like.
