Maintaining Programs

This chapter describes Emacs features for maintaining programs. The version control features (see Version Control) are also particularly useful for this purpose.

  • Change Log: Maintaining a change history for your program.
  • Tags: Go direct to any function in your program in one command. Tags remembers which file it is in.
  • Emerge: A convenient way of merging two versions of a program.
